# Telegram Welcome Bot

Bot ini menyambut member baru di grup Telegram dengan gambar dan tombol.

### Setup

1. Copy TOKEN dari BotFather.
2. Set environment variable di Render dengan key `TOKEN`.
3. Upload `welcome.jpg` ke root project.
4. Deploy bot.

### Fitur

- Welcome member baru
- Inline button `Rules` / `Info`
- /start command
- Echo text message
